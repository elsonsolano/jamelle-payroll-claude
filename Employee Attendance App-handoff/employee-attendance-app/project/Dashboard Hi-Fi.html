State machine (ring action)

  • none → tap logs Time In → in
  • in → tap logs Start Break → break
  • break → tap logs End Break → in
  • in (after scheduled end) → Time Out button → out
  • out → read-only until next shift

Ring progress = elapsedWork / scheduledHours. Pauses during break.

Data contracts

type ClockState =
  'none' | 'in' | 'break' | 'out';

interface DashboardData {
  user: { name, avatarUrl, role };
  today: Shift | null;
  tomorrow: Shift | null;
  current: {
    state: ClockState;
    clockedInAt?: string;
    breakStartedAt?: string;
    elapsedMinutes: number;
  };
  dailyQuote: { text, author };
}

Accessibility

  • Ring button: role="button", label describes current action (“Time in — shift starts 8 AM”).
  • Minimum tap target: 186×186 (hero), 44×44 (tab items).
  • Contrast: ring label on green-500 passes AA at 14px bold.
  • Live region: announce clock state changes.
  • Respect prefers-reduced-motion — disable live-dot pulse.
